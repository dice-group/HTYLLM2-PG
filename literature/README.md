# Literature Analysis

This folder contains the literature survey and analysis related to the HTYLLM2 Project.

## File Included

* `Literature_analysis.xlsx`
  Contains the collected research papers, summaries, methodology comparisons, and key findings for multilingual LLM literature review.

## Purpose

The goal of this literature analysis is to:

* study existing multilingual Large Language Models (LLMs)
* compare different approaches and architectures
* analyze datasets and evaluation methods
* identify research gaps for the project work
